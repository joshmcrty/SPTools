Batch Start Workflow

Sometimes you need to start a lot of workflows at once.

* indicates a required field.

1. Select Source List/Library and Items

Enter the display name or the GUID of the source list (e.g. “Announcements” or “{A123AB12-AB1A-12A1-1234-ABC1A1234A12}”).

Enter the path to the site or sub-site where the source list is located (e.g. “http://Server/Site”).

Enter the CAML Query to select the items you want to duplicate. By default this selects all items.

Click the Preview button to view all of the items that will have the workflow started. This will not perform the workflow.

2. Select Workflow

Enter the display name of the workflow to initiate on the list/library items.

3. Start Workflows

Warning: Make sure your source list/library is set properly, the correct workflow has been selected, and you've run a preview to ensure that the correct items will have a workflow initiated. Once this process begins it can only be stopped by closing your browser tab/window.
